Тестовое задание от Aikamsoft

Необходимо разработать приложение, предоставляющее сервис работы с данными в БД.
По аргементам командной строки определить тип операции:
search (Поиск по критериям)
stat (Статистика за период)
На вход и выход дается Json файл
Пример командной строки: Main search input.json output.json;

Используется: Java 8, PostgreSQL 9.6, Maven
Среда разработки: Inteliji Idea

База данных: Дамб находиться в папке PostgreSql.
Создается три таблицы:
customers(id, firstname, lastname)
products(id, name, price)
purchases(id, date, customer_id, product_id)

Java:
В программе есть логгер slf4j настроенный на консоль.
В программе для подключение к БД используется JDBC, конфигурация входа пишеться в config.properties

На операцию search пример входного файла представлен в input2.json.
На stat соответсвенно input.json.

В случае возникновенния ошибок - В консоль будут выводиться логи.

Если ошибка связанна c данными в input.json, то в output.json будет выводиться ошибка.
Если ошибка с отдельным критерием поиска, то в результат поиска будет выведена ошибка.
